Coding Everest

This is my personal website.

The theme is built with 100% CSS and HTML. I built it as a challenge to see how far I could get with standard front-end tools. It's also a project to, literally, code Mt. Everest.

Topics I want to explore with this project:

  • Advanced CSS3, benefits & limits, drawbacks and quirks (particularly cross-browser) for illustration and animation
  • Cross-browser compatibility (CSS3)
  • Landscape as responsive design, without detracting too much from content
  • Continual motion in website design
  • Impact of dynamic elements in design (changing time of day, weather)
